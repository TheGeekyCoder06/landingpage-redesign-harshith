Welcome to the **Landing Page Redesign Sprint**!  
This repository contains a very basic Next.js + TailwindCSS landing page.

### Task: Redesign and improve the landing page in ANY way you want.

## What You Can Improve

You are free to enhance anything, including:

- UI/UX improvements
- Layout redesign
- Add animations / transitions
- Better responsiveness
- Replace fonts & colors
- Add a better navigation bar or footer
- Add illustrations or hero graphics
- Add components (cards, CTA, testimonials, pricing, etc.)
- Improve accessibility and SEO
- Add dark mode
- ANYTHING that makes this look like a modern, beautiful landing page

Be as creative as you want!

## Tech Stack

- Next.js (latest, App Router)
- TailwindCSS
- TypeScript

## How to Contribute

1. **Fork** this repository
2. **Clone** the repository
3. Code in the forked repository
4. **Deploy** the page for ease of access
5. Submit the forked repository link on the website **when done**

## Getting Started

- Fork the repository
- Clone the repository: `git clone <forked-repository-link>`
- Install dependencies: `npm install`
- Run dev server: `npm run dev`
- Open the localhost on browser: http://localhost:3000
